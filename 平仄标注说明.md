# 平仄标注说明

## 方法概述

基于 **平水韵（106韵部）** 中古音系统为每首诗标注每个字的平仄。平水韵将汉字分为平声、上声、去声、入声四类，其中平声为"平"，上声、去声、入声为"仄"。

## 数据来源

使用开源 Python 包 `pingshui-rhyme` 内置的平水韵字表（原始数据来自维基文库）。

- 包名: `pingshui-rhyme` (v0.20)
- 项目地址: https://github.com/jkak/pingShuiYun
- 字典文件: `organized_ping_ze_rhyme_dict.json`

## 处理流程

```
平水韵字典（繁体8,564字）
    │
    ├─ opencc 繁体→简体（合并后8,375字）
    │     └─ 简繁合并冲突：20字（少数派覆盖）
    │
    ├─ 手动补充缺失高频字
    │     └─ 溪、疏、迹、啸、剑、猿等 + 常见入声字
    │
    ├─ 保存为 data/tone_dict.json（8,379字）
    │
    └─ 遍历每首诗每个字 → 查字典 → 输出序列
          └─ 保存为 data/tone.txt
```

## 平仄编码

| 编码 | 含义 |
|------|------|
| `ping` | 平声（平水韵上平声30韵+下平声30韵） |
| `ze` | 仄声（上声29韵+去声30韵+入声17韵） |
| `unk` | 未知（不在字典中的字符） |

`data/tone.txt` 中每行对应一首诗，空格分隔的平仄序列：

```
ping ping ze ze unk ping ze ze ...（第1首诗）
ze ze ping ping ze ...（第2首诗）
```

## 执行代码

```python
import json

# 加载平仄字典
with open('data/tone_dict.json', encoding='utf-8') as f:
    tone_dict = json.load(f)

# 读取诗句
with open('data/peot.txt', encoding='utf-8') as f:
    lines = f.read().split('\n')

# 生成平仄序列
tone_sequences = []
for poem in lines:
    tones = []
    for c in poem:
        if c in tone_dict:
            tones.append(tone_dict[c])  # 'ping' or 'ze'
        else:
            tones.append('unk')
    tone_sequences.append(' '.join(tones))

# 保存
with open('data/tone.txt', 'w', encoding='utf-8') as f:
    f.write('\n'.join(tone_sequences))
```

## 关于入声字

入声是平仄标注中最关键的难点。入声字在现代普通话中已消失（"入派三声"），分散到一二三四声里。例如：

| 字 | 现代普通话 | 中古音 | 实际平仄 |
|---|-----------|--------|---------|
| 一 | yī（一声） | 入声质韵 | **仄** |
| 白 | bái（二声） | 入声陌韵 | **仄** |
| 节 | jié（二声） | 入声屑韵 | **仄** |
| 独 | dú（二声） | 入声屋韵 | **仄** |
| 石 | shí（二声） | 入声昔韵 | **仄** |
| 黑 | hēi（一声） | 入声职韵 | **仄** |

如果用现代普通话声调标平仄，这些字会误标为平声。使用平水韵字典可正确处理。

## 已知局限

1. **简繁合并冲突**：20 个简化字对应多个不同平仄的繁体字（如"发"对应"發"和"髮"），当前取后出现的覆盖前者。
2. **多音字未处理**：部分字在不同语境下平仄不同（如"教"在"教学"中为平，在"教育"中为仄），当前只取一种。
3. **扩展区汉字缺失**：CJK 扩展区（U+3400+）的汉字未收入平水韵字典。
