### 数据流

```
唐诗 JSON (data/tang/) ─→ _parse_tang_to_records ─┐
                                                   ├──→ _detect_genre → _quality_flag
CCPC JSONL (CCPC/)    ─→ _parse_ccpc_to_records  ─┘         │
                                                              ↓
                                                     _dedup_with_split_protection
                                                              │
                                                              ↓
                                                     _split_tang_and_merge
                                                      80/10/10 + CCPC合并
                                                              │
                                                              ↓
                                                     _is_high_confidence
                                                              │
                                                              ↓
                                                     _apply_char_mapping
                                                      □→PLACEHOLDER, 低频→UNK
                                                              │
                                                              ↓
                                          data/processed/{train,valid,test,all}.jsonl
                                          data/processed/{data_stats,dedup_report,review_samples}.jsonl
                                          docs/data_notes.md
```

### 体裁规则

| 规则 | 分类 |
|------|------|
| 4句×5字 | WUJUE (五言绝句) |
| 4句×7字 | QIJUE (七言绝句) |
| 8句×5字 | WULV (五言律诗) |
| 8句×7字 | QILV (七言律诗) |
| 其他 | OTHER |
| CCPC 已知为绝句，仅区分五言/七言 | |

### 去重优先级

1. CCPC test — 从所有训练和验证来源移除重复（634 条）
2. CCPC valid — 从所有训练来源移除重复（528 条）
3. 训练内部重复 — 只保留一条，tang 优先于 CCPC train（7,032 条）

### 字频映射

- `□` → `[PLACEHOLDER]`（可作为输入；target=PLACEHOLDER 忽略损失；生成禁止）
- 出现次数 ≤ 3 的低频汉字 → `[UNK]`（默认禁止输出）

### 特殊 Token 约定

| Token | ID | 规则 |
|-------|----|------|
| PAD | 0 | 禁止正文输出 |
| UNK | 1 | 默认禁止输出 |
| SOP | 2 | 禁止正文输出 |
| EOP | 3 | **必须允许输出** |
| PLACEHOLDER | — | 可输入；target忽略损失；生成禁止 |
